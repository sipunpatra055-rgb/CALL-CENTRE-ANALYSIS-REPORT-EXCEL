## Call Center Performance Dashboard
## 📌 Project Overview
This project features an interactive dashboard designed to analyze and visualize call center performance metrics. The goal of this analysis is to provide actionable insights into customer satisfaction (CSAT), communication channel effectiveness, geographical trends, and service level agreement (SLA) compliance.

By analyzing this dataset, the dashboard helps identify key drivers of customer sentiment and operational bottlenecks, making it a valuable tool for data-driven decision-making in customer service management.

## 📊 Key Features & Insights
The dashboard provides a comprehensive view of operations through several key metrics and visualizations:
**Executive KPIs: Tracks high-level metrics including Total Customers (70) and overall Average CSAT score (5.3).
**Channel Analysis: Compares interaction volumes and CSAT scores across different support channels (Call-Center, Chatbot, Email, and Web).
**Geographical Trends: Visualizes Statewide Average CSAT and Statewide Call Handling volumes, easily identifying high-traffic regions like California, Texas, and Florida.
**Root Cause & Sentiment: Breaks down the primary reasons for customer contact (e.g., Billing Questions, Payments, Service Outages) and maps the overall distribution of customer sentiment (from Very Negative to Very Positive).
**Operational Efficiency: Tracks Response Counts against internal SLAs to monitor how often the team responds within the target timeframe versus falling below SLA.
**Interactive Slicers: Includes interactive filters for Channel and State, allowing users to dynamically drill down into specific data segments for targeted EDA.

## 🛠️ Tools & Techniques Used
**Data Visualization: Created dynamic charts (Bar, Column, Pie) to represent complex categorical and numerical data.
**Data Aggregation: Utilized grouping and summarizing techniques (such as Pivot Tables) to calculate averages, counts, and performance metrics.
